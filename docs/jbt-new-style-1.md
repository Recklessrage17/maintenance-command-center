# JBT New Style 1

JBT New Style 1 is the official visual style for the current Maintenance Command Center (MCC) website shell.

## Purpose

This style defines the first MCC look: a clean maintenance department command center that is modern, readable, and practical without being overdesigned.

## Core Visual Direction

- Use a dark navy command center background as the primary surface.
- Use cyan and blue maintenance highlights for active states, status indicators, borders, and important interface accents.
- Keep the interface clean and operational, with a maintenance department feel instead of a consumer app feel.
- Prefer clarity, readable labels, and obvious controls over decorative complexity.
- Keep the design modern but not overdesigned.

## Layout Pattern

- Use sidebar navigation for primary MCC modules.
- Keep the top header visible and easy to scan.
- Present dashboard content in large rounded cards.
- Use tablet- and phone-friendly spacing so the app remains usable on shop-floor devices.
- Make labels easy to read at a glance.
- Make clickable buttons large enough for tablet and touch use.

## Dashboard Cards

Dashboard cards should be:

- Large and rounded.
- Easy to scan from a short distance.
- Clear about what module, metric, or placeholder they represent.
- Styled with dark surfaces, subtle borders, and cyan/blue emphasis where needed.
- Useful and restrained, avoiding unnecessary animation or visual clutter.

## Navigation

Sidebar navigation should:

- Clearly list MCC modules.
- Use large clickable buttons.
- Show the active module with cyan/blue emphasis.
- Stay readable and usable on tablets and phones.
- Keep module labels simple and direct.

## Safety and Product Boundaries

- MCC remains on port `4273`.
- MCC is the current daily-use maintenance system.
- Retired tracker workflows should not appear in normal MCC UI.

## Design Checklist

When adding new MCC screens or components, confirm that they follow JBT New Style 1:

- Dark navy command center background.
- Cyan/blue maintenance highlights.
- Sidebar-first navigation model.
- Large rounded dashboard cards where card layouts are appropriate.
- Tablet/phone friendly spacing.
- Clean maintenance department look.
- Modern but not overdesigned.
- Easy labels and large clickable buttons.
- MCC-native wording throughout normal screens.
- MCC port `4273` remains unchanged.
