# Maintenance Command Center

Maintenance Command Center is the new main dashboard/hub for the maintenance department.

MIT3 / Maintenance Inventory Tracker 3 stays protected and working in its own repo.

## Planned Modules

- Dashboard
- Inventory
- Preventive Maintenance
- Assets
- Work Orders
- Requisitions
- Vendors
- Locations
- Documents / Prints
- Reports
- Settings

## Safety Rule

Do not break MIT3. Inventory integration comes later after the MCC shell is stable.
